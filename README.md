# TSP
Solving Traveling Salesman Problem by GRASP-ILS-RVND

## Compilação
```
$ g++ main.cpp readData.cpp -o <nome executavel>
```

## Execução
```
$ ./<nome executavel> <arquivo entrada> [seed]
```
